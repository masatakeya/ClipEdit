# ClipEdit

スマートフォンブラウザでの利用に特化した、シンプルで強力なクリップボードテキストエディタです。
他のアプリからコピーしたテキストを一時的に編集・整形して、再びコピーするために設計されています。

## 特徴

- **📋 クリップボード連携**: ボタン一つでペースト・コピーが可能。
- **📱 モバイルファースト**: スマホでの操作性を重視したUI。
- **💾 自動保存**: 入力内容は自動的に保存され、ブラウザを閉じても消えません。
- **↩️ 編集支援**: Undo/Redo、全削除、全選択機能を完備。
- **🔍 検索・置換**: 高度な検索・置換機能を搭載（正規表現対応準備中）。
- **📊 統計情報**: 文字数・行数をリアルタイムで表示。
- **🔒 プライバシー重視**: データはすべてブラウザ内に保存され、外部サーバーには送信されません。

## 使い方

1. **ペースト**: 「Paste」ボタンを押すと、クリップボードの内容がエディタに貼り付けられます。
2. **編集**: テキストエリアでテキストを自由に編集します。
3. **コピー**: 「Copy」ボタンを押すと、編集した内容がクリップボードにコピーされます。

## 実行方法（ローカル）

このアプリはES Modulesを使用しているため、ローカルサーバー経由で実行する必要があります。

### Pythonを使用する場合

```bash
python -m http.server 8000
```

ブラウザで [http://localhost:8000](http://localhost:8000) にアクセスしてください。
